## Installation

```bash
$ nvm use
$ npm i
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Lint & Prettier, Test & Coverage

```bash
# lint tests
$ npm run lint

# prettier write
$ npm run format

# unit tests
$ npm run test

# test coverage
$ npm run test:cov

# test e2e - Start mosquitto broker in the background at localhost:1883!
$ npm run test:e2e
```

## Generating documentation Swagger & AsyncAPI

```bash
# generate docs
$ npm run docs
```
